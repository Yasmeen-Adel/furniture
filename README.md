# Furniture App 🛋️

A simple Flutter app for browsing and buying furniture items.

## Features:
- Home screen with categories (Sofa, Desk, etc)
- Product cards with image, description, and price
- Theming and custom color support

## Getting Started:
To run the app:
```bash
flutter pub get
flutter run
